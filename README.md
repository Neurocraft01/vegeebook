# Vegeebook Next.js Project

This is a Next.js project for Vegeebook Solutions (OPC) Private Limited.

## Getting Started

1.  Open the terminal in VS Code.
2.  Navigate to the project directory:
    ```bash
    cd vegeebook-nextjs
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Run the development server:
    ```bash
    npm run dev
    ```
5.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

*   `app/page.tsx`: Landing Page
*   `app/about/page.tsx`: About Us (Story, Vision, Team)
*   `app/products/page.tsx`: Products & Services
*   `app/contact/page.tsx`: Contact Form
*   `app/faq/page.tsx`: FAQ
*   `app/components/`: Reusable components like Navbar and Footer

## Technologies Used

*   Next.js 14 (App Router)
*   React
*   Tailwind CSS
*   Lucide React (Icons)
*   TypeScript
